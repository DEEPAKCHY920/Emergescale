## 🚀 Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/DEEPAKCHY920/Emergescale.git
cd Emergescale

# 2. Install dependencies
npm install

# 3. Start the server
node server.js

# 4. Open in browser
# → http://localhost:3000
```

> **Tip:** Install `nodemon` for hot-reloading during development — `npx nodemon server.js`
